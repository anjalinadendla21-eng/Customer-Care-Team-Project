# Customer Care Registry

## Project Description
Customer Care Registry is a Flask-based web application developed to manage customer complaints efficiently. It allows users to add, update, search, and delete customer complaints while tracking their status through an interactive dashboard.

## Features
- Add Customer Complaint
- Edit Customer Details
- Delete Customer Record
- Search by Name or Phone
- Dashboard with Complaint Statistics
- SQLite Database Integration
- Responsive Bootstrap Interface

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS
- Bootstrap

## How to Run

1. Install Flask

```
pip install -r requirements.txt
```

2. Run the application

```
python app.py
```

3. Open the browser

```
http://127.0.0.1:5000
```
